# TableCRM Mobile Order Form

Мобильное веб-приложение для создания продажи / заказа через API TableCRM.

Тестовое задание на позицию React Frontend Developer.

## 🚀 Описание проекта

Приложение представляет собой мобильную форму оформления заказа в формате WebApp.

Пользователь вводит токен кассы, после чего получает доступ к созданию продажи:

- поиск клиента по телефону
- выбор счета
- выбор организации
- выбор склада
- выбор типа цен
- добавление товаров
- расчет суммы
- создание продажи
- создание и проведение продажи

Интерфейс реализован с упором на mobile-first UX.

## 📦 Установка проекта

```bash
git clone <repo_url>
cd tablecrm-mobile-order
npm install
````

---

## ▶️ Запуск в режиме разработки

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:

```bash
http://localhost:3000
```

или для Vite:

```bash
http://localhost:5173
```

---

## 🏗 Сборка production версии

```bash
npm run build
npm run start
```

---

## 🔑 Использование

1. Откройте приложение
2. Введите токен кассы TableCRM
3. Дождитесь загрузки данных
4. Заполните форму заказа
5. Нажмите:

* **Создать продажу**
* **Создать и провести**
